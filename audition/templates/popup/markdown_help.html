Markdown Help

Markdown is an easy to read and write formatting system that allows you to format text in text fields on this site.

Examples are below

Text Decoration

Easily embolden or italicise text:

**Bold Text**
*Italic Text*

Output:

{% filter markdown %} **Bold Text** *Italic Text* {% endfilter %}

New Lines

Make sure you leave a blank line before text that you want to put on a new line.

Line One

Line Two

Output:

{% filter markdown %} Line One Line Two {% endfilter %}

Headers

Underline lines to make them headers:

Top Level Header
================
Sub Level Header
----------------

Output:

{% filter markdown %} Top Level Header ================ Sub Level Header ---------------- {% endfilter %}

Lists

Lists are easy! And can be ordered or unordered:

- Unordered Item 1
- Unordered Item 2
- Unordered Item 3

1. Ordered Item 1
2. Ordered Item 2
3. Ordered Item 3
{% filter markdown %} - Unordered Item 1 - Unordered Item 2 - Unordered Item 3 {% endfilter %} {% filter markdown %} 1. Ordered Item 1 2. Ordered Item 2 3. Ordered Item 3 {% endfilter %} {% endblock %}
